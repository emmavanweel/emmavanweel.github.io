kernel density map

This map uses vector data of all the trees in Utrecht to analyze the kernel density of the trees in the city.

This map, created from vector data of all the trees in Utrecht, is used to analyze the city's tree density. The map-making process began with uploading the data to the world map in ArcGIS Pro via ArcGIS Online. Next, the municipal boundaries were added. The data was then analyzed by creating a kernel density map, which illustrates the tree density across all of Utrecht.

This new data can be used to address problems related to Sustainable Development Goal 15: Life on Land. By providing insights into the distribution of tree density across Utrecht, it helps identify areas with low tree density where new trees should be planted. This can enhance urban green spaces, improve air quality and contribute to biodiversity. In addition, the data highlights areas with high tree density that should be conserved to protect the trees.
